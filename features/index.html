Frequently Asked Questions

Will a computer be provided?
No, you must provide a computer with internet access, and a USB port.
Does my computer need to run Windows?
No, any operating system with a modern web browser and a USB port will work.
Does my computer need a web camera?
No, although you may use the Play Store Fun system without a web camera, we strongly encourage having one in order to take pictures of the items in your store.
Where do I get items to stock my play store?
This is the fun part! You can use anything that your imagination can come up with. Do you want to setup a grocery store? Recycled food containers are perfect. Do you want to setup a pet store? Grab some stuffed animals and let the fun begin. We include barcodes that can be applied to items that might not already have a barcode on them.
